Custom created Django 'Native' web-app. Native - is UI(User Interface) based develop only on Django-templates(MVT) without any frontend framework(react/vue/angular).

This simple project contains:

  • django: based on MVT (Model-View-Template) pattern
  • django: CRUD(Create-Read-Update-Delete)-like endpoints
  • django: with 'static' data and external 'templates' folder
  • django: custom templatetags, filters, context-processors
  • django-template: scalable extends/include structure
  • django-template: Paginator
  • production: with all cmd+sh scripts (+ comments)
  • bootstrap css styles + js scripts
  • custom css styles (with font-family: 'Zen Kaku Gothic Antique')
  • and more...
